## Installation

1. Download the latest `.crx` file from this repo (or build your own).  
2. Open Chrome and go to `chrome://extensions`.  
3. Enable **Developer mode** (top-right).  
4. Drag and drop the `.crx` file into Chrome.  
5. Boom! Extension installed.
